Let's begin with upgrading everything at the first boot

#+begin_src sh :results verbatim :dir /sudo::

#+end_src

#+begin_src sh :results verbatim :dir /sudo::

#+end_src

* SSH Setup
** From Old Setup                                                  :dropbox:
#+begin_src sh
  ln -s $HOME/Dropbox/.ssh $HOME/.ssh
#+end_src

** From Scratch                                                    :scratch:
*** Generate
#+begin_src sh
  ssh-keygen -t rsa -b 4096 -C "cestdiego@gmail.com"
  eval "$(ssh-agent -s)"
  ssh-add ~/.ssh/id_rsa
#+end_src
*** Copy it
#+begin_src sh :results verbatim :dir /sudo::
  sudo apt-get install xclip
#+end_src

#+begin_src sh
  xclip -sel clip < ~/.ssh/id_rsa.pub
#+end_src
*** Test
#+begin_src sh
  ssh -T git@github.com
#+end_src

* Emacs
** Installation                                                      :build:
*** Build Dependencies for Emacs 24.5

#+begin_src sh :results verbatim :dir /sudo::
  sudo apt-get build-dep emacs24 -y
#+end_src

*** Getting the source code
#+begin_src sh
  wget http://ftp.gnu.org/gnu/emacs/emacs-24.5.tar.gz
  tar -xf emacs-24.5.tar.*
  cd emacs-24.5
#+end_src
*** Building it
#+begin_src sh
  ./configure
  make
#+end_src

*** Install it
#+begin_src sh :results verbatim :dir /sudo::
  sudo make install
#+end_src
